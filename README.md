# Portfolio

A modern, full-stack portfolio website built with HTML, CSS, JavaScript, and Node.js.

## Features
- **3D Tilt Cards & Parallax Animations**
- **Dynamic Backgrounds & Night/Day Mode**
- **Working Contact Form** (Node.js + Nodemailer backend)
- **Responsive Design**

## Setup
1. Clone the repository
2. Run `npm install`
3. Set up your `.env` file with email credentials
4. Start the server: `node server.js`
